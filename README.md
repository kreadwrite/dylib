<h1 align="center">PXTok</h1>

<p align="center">Твик для TikTok на джейлбрейке. Личная сборка, без рекламы и лишних ограничений.</p>

---

## Установка

Открой **TikTok → Профиль → ≡ → Settings and privacy → Account** — в самом верху раздела появится пункт **PXTok** с фирменной иконкой. Там один экран со всеми переключателями, разбитыми по разделам ниже.

---

## Feed — лента

| Функция | Что делает |
|---|---|
| Hide Ads | Скрывает рекламные вставки в ленте |
| Download Button | Добавляет кнопку скачивания видео по долгому нажатию |
| Share Sheet | Восстанавливает/настраивает системное меню "Поделиться" |
| Remove Watermark | Убирает водяной знак TikTok со скачанных видео |
| Show/Hide UI Button | Кнопка для скрытия интерфейса поверх видео (чистый просмотр) |
| Stop Playback | Останавливает автовоспроизведение при определённых условиях |
| Auto Play Next Video | Автоматически включает следующее видео по завершении текущего |
| Show Progress Bar | Показывает полосу прогресса воспроизведения на видео |
| Transparent Comments | Делает панель комментариев полупрозрачной |
| Show Usernames | Показывает реальные юзернеймы там, где TikTok прячет их за никнеймом |
| Disable Sensitive Content Warning | Убирает предупреждение под "деликатным" контентом |
| Disable Warnings | Убирает прочие предупреждающие плашки под видео |
| Disable Live Streaming | Скрывает прямые эфиры из ленты полностью |
| Skip Recommendations | Автоматически пролистывает карточки "рекомендуем подписаться" |
| Upload Region | Показывает регион, в котором было выпущено видео |

## Profile — профиль

| Функция | Что делает |
|---|---|
| Profile Save | Кнопка сохранения аватарки пользователя |
| Profile Copy | Копирование информации профиля (никнейм, био, статистика) |
| Video Like Count | Показывает точное число лайков на видео |
| Video Upload Date | Показывает дату загрузки видео |

## Confirm — подтверждения

| Функция | Что делает |
|---|---|
| Like Confirmation | Просит подтвердить перед постановкой лайка |
| Like Comment Confirmation | Подтверждение перед лайком комментария |
| Dislike Comment Confirmation | Подтверждение перед дизлайком комментария |
| Follow Confirmation | Подтверждение перед подпиской на профиль |

## Other — разное

| Функция | Что делает |
|---|---|
| Always Open Safari | Внешние ссылки всегда открываются в Safari, а не во встроенном браузере |
| Enable Fake Changes | Включает визуальную подмену счётчиков (см. ниже) |
| Fake Follower Count | Свой текст вместо реального числа подписчиков (виден только тебе) |
| Fake Following Count | Свой текст вместо числа подписок (виден только тебе) |
| Fake Verified | Добавляет синюю галочку верификации в твоём интерфейсе |
| Extended Bio | Увеличивает лимит символов в био |
| Extended Comment | Увеличивает лимит символов в комментарии |
| Padlock | Блокирует запуск приложения экраном пароля |

## Region — регион

| Функция | Что делает |
|---|---|
| Enable Region Changing | Включает смену определяемого региона |
| Select Country | Выбор страны из списка для подмены региона |

## Live Button Function — кнопка эфиров

| Функция | Что делает |
|---|---|
| Enable | Включает переопределение действия кнопки "Live" |
| Select Action | Выбор, что будет делать кнопка вместо перехода к эфирам |

## Playback Speed — скорость видео

| Функция | Что делает |
|---|---|
| Enable | Включает изменение скорости воспроизведения |
| Select Speed | Выбор скорости (0.5x–2x) |

## Developer — разработка

| Функция | Что делает |
|---|---|
| Flex Mode | Открывает FLEX-инспектор для отладки интерфейса приложения |
| Ссылки автора | Github/X/донат — на разработчика оригинальной базы |

## PXTok Extra — то, что добавлено сверх базы

| Функция | Что делает | Статус |
|---|---|---|
| Автосохранение избранного | При добавлении видео в избранное тихо сохраняет копию на устройство — на случай если автор удалит видео | ⚠️ нужно проверить имя иконки кнопки |
| Перевод описания | Кнопка в меню видео открывает Google Translate с текстом описания | ✅ |
| Перевод комментариев | Кнопка под комментарием переводит его текст через Google Translate | ⚠️ имя поля текста нужно подтвердить class-dump'ом |
| Копия коммента без ника | Долгое нажатие на комментарий копирует только текст, без имени автора | ⚠️ имя поля текста нужно подтвердить class-dump'ом |
| Скрыть кнопку «+» | Прячет кнопку создания на нижней панели | ⚠️ best-effort, ищет по accessibility-label |
| Скрыть значки уведомлений | Убирает красные точки на вкладках "Друзья"/"Входящие" | ⚠️ best-effort, ищет по имени класса |
| Отключить подсказки | Скрывает всплывающие подсказки в комментариях | ⚠️ best-effort, ищет по имени класса |
| Скрыть мою «нечиталку» | Не отправлять статус "просмотрено" в личных сообщениях | ⛔ пока не реализовано, нет доступа к IM-классам |
| OLED-клавиатура | Форсирует тёмную клавиатуру во всём приложении | ✅ |
| Скрыть панель эмодзи | Прячет панель эмодзи в поле ввода комментария | ⚠️ best-effort, ищет по имени класса |
| Авто-раскрытие веток | Автоматически разворачивает ответы на комментарии | ⚠️ зависит от английского текста кнопки |
| Язык настроек PXTok | Переключает вкладку PXTok Extra между русским и английским | ✅ |
| Статистика использования | Свой экран: видео за сегодня/всего, время в приложении, число сессий | ✅ |
| Регулировка вибрации | Включает выбор силы виброотклика ниже | ✅ |
| Сила вибрации | Слабая / Средняя / Сильная / Выкл | ✅ |
| Скрыть значок «В эфире» | Убирает индикатор прямого эфира на аватарках в ленте | ✅ |
| Прозрачная навигация | Убирает заливку и тень с верхней панели интерфейса | ✅ |
| Призрак: экран приватности | Размывает содержимое TikTok в переключателе приложений iOS, пока ты его не открыл заново — рядом никто не подсмотрит | ✅ |
| Призрак: очистка буфера | Через 30 секунд стирает текст, скопированный через PXTok (описание/комментарий), из буфера обмена | ✅ |
| Призрак: скрыть от Spotlight | Не даёт TikTok индексировать контент в системном поиске iOS | ✅ |

**Условные обозначения:** ✅ — использует реальные подтверждённые классы/системные API, должно работать сразу. ⚠️ — работает по общему шаблону/предположению об именах, может потребовать правки после первой сборки. ⛔ — только тумблер-заглушка, логика ещё не реализована.

## Сборка

```bash
make package FINALPACKAGE=1
```

Готовый `.deb` появится в `packages/`. Ставится через Sileo/Zebra на джейлбрейкнутое устройство, либо через `make package install` с настроенным `THEOS_DEVICE_IP`.
